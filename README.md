# Tehno News — Персональный новостной агрегатор

> **Статус:** Активная разработка (MVP)
> **Стек:** HTML5, CSS3 (Flexbox), Vanilla JS (планируется)
> **Цель:** Создать минималистичный, быстрый и адаптивный сайт для чтения новостей с фокусом на технологии и бизнес.

## 📜 О проекте

**Tehno News** — это прототип новостного сайта, разработанный для демонстрации навыков верстки и понимания структуры веб-приложений. Проект имитирует работу новостного агрегатора, где контент разбит на тематические ленты (как показано в документации CMS), а главная страница представляет собой динамичную ленту новостей с карточками.

В отличие от тяжелых фреймворков, этот сайт построен на чистом HTML и CSS. Это гарантирует:
*   **Мгновенную загрузку:** Отсутствие тяжелых скриптов позволяет странице открываться практически мгновенно даже на мобильных устройствах с медленным интернетом.
*   **Доступность:** Семантическая верстка (теги `<header>`, `<main>`, `<article>`, `<footer>`) делает сайт понятным для поисковых роботов (Яндекс, Google) и скринридеров.
*   **Простую поддержку:** Код легко читать и редактировать даже новичку.

## 🎯 Основные возможности

На текущем этапе проект реализует следующий функционал:

*   **Адаптивная лента новостей:** Главная страница (`index.html`) отображает карточки новостей в формате «Иконка + Категория + Заголовок + Анонс (2 строки) + Дата».
*   **Навигация по разделам:** Пользователь может переключаться между главной страницей, разделом «Все новости», категорией «Техно-новости» и страницей «О нас».
*   **Интерактивность:** Карточки новостей на главной странице имеют эффект плавного подъема при наведении курсора. Текст анонсов автоматически обрезается, если он слишком длинный (используется CSS свойство `-webkit-line-clamp`).
*   **Единая стилистика:** Все страницы используют общую цветовую палитру и типографику, создавая ощущение целостного продукта.
*   **Верификация:** В мета-тегах страниц предусмотрен код для подтверждения прав на сайт в Яндекс.Вебмастере.

## 🗂 Структура проекта

Проект состоит из набора статических файлов, расположенных в одной директории:

| Файл | Назначение |
| :--- | :--- |
| `index.html` | **Главная страница.** Содержит основную ленту новостей, заголовок проекта и блок действий с кнопками. Сюда встроен актуальный код верификации Яндекса. |
| `about-us.html` | **Страница «О нас».** Рассказывает о миссии проекта, команде и помощи малому бизнесу. Содержит ссылки на другие разделы. |
| `news.html` | **Страница «Все новости».** Представляет собой список всех доступных новостей в упрощенном виде (заголовок + метаданные). |
| `tehno-news.html` | **Страница «Техно-новости».** Отдельный раздел, посвященный исключительно технологиям, гаджетам и ИИ. |
| `README.md` | **Этот файл.** Документация проекта. |

## 🎨 Дизайн и UX

Дизайн сайта выполнен в стиле **«Clean News»** (Чистые новости):
*   **Цветовая гамма:** Спокойный темно-синий (`#2c3e50`) для заголовков и акцентов, оранжевый (`#e67e22`) для выделения категорий, светло-серый фон для комфорта глаз.
*   **Типографика:** Используются системные шрифты (`Segoe UI`, `Roboto`, `Arial`), что обеспечивает корректное отображение на любых устройствах без необходимости подгружать тяжелые шрифты с Google Fonts.
*   **Мобильная адаптация:** Благодаря использованию Flexbox и `max-width` для контейнера, сайт корректно отображается как на десктопах, так и на смартфонах.

## 🛠 Как запустить проект

Поскольку это статический сайт, для его запуска не требуется серверная часть (Node.js, PHP и т.д.) или сложная сборка.

### Вариант 1: Локальный запуск (на своем компьютере)

1.  Убедитесь, что у вас есть папка с файлами проекта (`index.html`, `about-us.html` и т.д.).
2.  Найдите файл `index.html` в этой папке.
3.  Дважды кликните по нему. Сайт откроется в вашем браузере по протоколу `file://`.
4.  Проверьте работу кнопок навигации и hover-эффектов.

### Вариант 2: Публикация в интернете (GitHub Pages)

Чтобы сайт стал доступен по ссылке в интернете:

1.  Зарегистрируйтесь на GitHub (если нет аккаунта).
2.  Создайте новый репозиторий (например, `tehno-news`).
3.  Загрузите все файлы из вашей папки в этот репозиторий.
4.  Зайдите в настройки репозитория (**Settings**) → раздел **Pages**.
5.  В пункте **Branch** выберите `main` (или `master`) и нажмите **Save**.
6.  Через 1–2 минуты сайт будет доступен по адресу: `ваш-ник.github.io/tehno-news`.
